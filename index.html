PyNames

Type a Keyword to get started! arrow_upward

PyNames is a simple tool to help name python projects so they contain "py". Enter a keyword in the search box to get a list of potential python project names with various info about how the names were derived. This tries to bias towards names that are less contrived, but only uses a set of simple heuristics.
